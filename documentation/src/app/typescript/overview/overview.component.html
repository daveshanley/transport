Transport for TypeScript

This implementation is designed to operate in the Browser (sorry node.js folks). However it is framework agnostic, and will work along-side any UI framework you want to implement.

Most of this documentation is focused around using Angular as a UI framework, however we have added in a few React examples as well, just remember that thesd are just examples and Transport does not care which framework you would like to run. It's directly accessible from the window object, if you don't even want to use a framework.
